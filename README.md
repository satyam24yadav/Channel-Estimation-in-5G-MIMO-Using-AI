#  Channel Estimation in 5G MIMO using AI

##  Abstract
This project presents an AI-based channel estimation technique for 2x2 5G MIMO systems using MATLAB. It improves estimation accuracy compared to traditional LS and MMSE methods.

---

##  Problem Statement
Traditional channel estimation techniques in wireless communication suffer from noise and reduced accuracy. This project uses Artificial Neural Networks (ANN) to improve estimation performance.

---

##  Methodology
1. Implemented LS & MMSE channel estimation
2. Generated training data using MATLAB
3. Trained ANN model for channel estimation
4. Evaluated performance using BER vs SNR

---

##  Technologies Used
- MATLAB  
- Artificial Neural Networks (ANN)  
- 5G MIMO Systems  

---

##  Project Structure
- A_LS_MMSE_Channel_Estimation.m → Traditional estimation  
- B_AI_Training_Data_Generation.m → Data generation  
- C_ANN_Channel_Estimation_Model.m → ANN model training  
- D_ANN_BER_vs_SNR.m → Performance analysis
